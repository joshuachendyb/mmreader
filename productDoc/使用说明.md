## 使用方式

### 方式1：直接打开（最简单）

1. 双击 `app.html`
2. 等浏览器自动打开
3. 选择 Markdown 文件即可使用

### 方式2：HTTP 服务器（推荐，避免跨域问题）

```bash
cd D:\2bktest\MDview\MermaidReader
python -m http.server 8080
# 浏览器打开 http://localhost:8080/app.html
```

### 方式3：使用 Node.js

```bash
cd D:\2bktest\MDview\MermaidReader
node -e "const http=require('http'),fs=require('fs');const s=http.createServer((q,r)=>{const p=q.url==='/'?'/app.html':q.url;fs.readFile('.'+p,(e,d)=>{if(e)r.writeHead(404),r.end();else{const e=p.split('.').pop();r.writeHead(200,{'Content-Type':{'html':'text/html','css':'text/css','js':'text/javascript'}[e]||'text/plain'})};r.end(d)}})});s.listen(8080,()=>console.log('http://localhost:8080/app.html'))"
```

## 功能特性

| 功能 | 说明 |
|------|------|
| **Markdown 渲染** | 使用 marked.js 解析 |
| **Mermaid 图表** | 支持流程图、时序图、甘特图、类图等 |
| **代码高亮** | 使用 highlight.js |
| **目录导航** | 自动提取标题，点击跳转 |
| **滚动高亮** | 滚动时自动高亮当前章节 |
| **深色模式** | 支持亮色/深色主题切换 |
| **字体调节** | 支持 80%-150% 字体大小调节 |
| **设置记忆** | 主题和字体设置自动保存 |

## 常见问题

### Q: 为什么不能双击打开？
A: 请用浏览器打开（Chrome/Edge/Firefox）

### Q: Mermaid 图表不显示？
A: 确保网络正常，首次需要加载库文件

### Q: 深色模式不生效？
A: 清除浏览器缓存后重试

### Q: 如何导出 PDF？
A: 在浏览器中按 Ctrl+P，选择"另存为 PDF"
