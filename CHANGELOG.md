## [v1.0.0] - 2026-02-12

### 新增

- ✅ 初始版本发布
- ✅ Go + Wails 桌面应用架构 (gobuild/)
- ✅ Rust + Tauri 架构 (rustbuild/)
- ✅ Web 版本 (web/)
- ✅ Markdown 文件读取功能
- ✅ Mermaid 图表渲染功能
- ✅ 表格渲染功能
- ✅ 便携版发布包

### 核心功能

- 📄 文件读取：通过文件选择器打开本地 Markdown 文件
- 📊 图表渲染：支持所有 Mermaid 图表类型
- 📈 表格渲染：Markdown 表格转 HTML 表格
- 🚀 便携版：无需安装，直接浏览器运行

### 技术特性

- 多架构实现：Go、Wails、Rust、Tauri
- Web版本轻量级设计
- 完整的开发文档和测试用例
